# Installation

## Prérequis

- Python 3.9+
- Docker
- Docker Compose

## Étapes d'installation

1. Clonez le dépôt :

    ```sh
    git clone <URL_DU_DEPOT>
    cd multilang_site
    ```

2. Créez et activez un environnement virtuel :

    ```sh
    python3 -m venv env
    source env/bin/activate
    ```

3. Installez les dépendances :

    ```sh
    pip install -r requirements.txt
    ```
